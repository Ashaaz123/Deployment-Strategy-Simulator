# 🚀 Deployment Strategy Simulator

A simple and interactive web application that demonstrates the concept of a **CI/CD Deployment Pipeline** using HTML, CSS, and JavaScript.

---

## 📖 Overview

The Deployment Strategy Simulator visualizes how an application moves through different deployment stages:

```text
Development → Testing → Production
```

Users can initiate a deployment process and observe the application's progression through each stage, simulating a real-world software deployment workflow.

## 🚀 How It Works

1. The application starts in the **Development** stage.
2. Clicking the **Start Deployment** button initiates the deployment process.
3. The application is automatically moved to the **Testing** stage.
4. After successful testing, it proceeds to the **Production** stage.
5. The deployment status is updated throughout the process.
